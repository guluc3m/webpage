Grupo de Arduino

En el grupo de Arduinos trabajamos la electrónica y los ordenadores embebidos con el hardware que nuestro nombre indica, Arduino.

Tenemos un repositorio en GitHub donde organizamos todos los proyectos del grupo y subimos nuestro material.

Actualmente este grupo no tiene un proyecto activo

Grupo de Hack3rs

Enfocado a la seguridad informática y criptografía, pasamos el rato investigando sobre protocolos de conexión y transmisión de datos, redes de ordenadores y encriptación y cifrado. Hacemos reuniones para aprender y compartir conocimientos sobre temas fijados anteriormente, y en nuestro grupo de Telegram enviamos recursos y material para reforzar.

Tenemos un repositorio de GitHub donde organizamos la temática y los proyectos internos.

Grupo de Juegos

El grupo de juego se encarga del desarrollo de videojuegos a través de software libre como Pygame o Godot. El proyecto actual es una novela gráfica al más puro estilo del profesor Layton, está siendo desarrollado con Renpy con Python y la librería Pygame.

Este es nuestro repositorio de GitHub donde podrás encontrar la situación actual del proyecto junto a más indicaciones a cerca del software empleado.

Grupo de MagIA

En el grupo de MagIA trabajamos diferentes técnicas relacionadas con la Inteligencia Artificial como las redes neuronales, Machine Learning, Deep Learning, etc.

Tenemos un repositorio en GitHub donde organizamos las tareas y temáticas, y administramos nuestros proyectos.
