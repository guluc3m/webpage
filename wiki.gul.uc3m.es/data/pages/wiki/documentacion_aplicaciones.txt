  * [[http://reciclado100.blogspot.com/2007/08/soundconverter-conversor-de-audio-para.html|SoundConverter, conversor de audio para Gnome]]. ===== Bash =====
  * [[http://www.tldp.org/LDP/Bash-Beginners-Guide/html/|Bash Guide for Beginners]].
  * [[http://www.cyberciti.biz/nixcraft/linux/docs/uniqlinuxfeatures/lsst/|Linux Shell Scripting Tutorial - A Beginner's handbook]]. This Linux Shell Scripting Tutorial is designed for people who wish to learn the fundamentals of shell scripting/programming. It also contains an introduction to power tools such as awk and sed.
  * [[http://tldp.org/LDP/abs/html/|Advanced Bash-Scripting Guide]]. An in-depth exploration of the art of shell scripting.

  * [[http://www.ibm.com/developerworks/library/l-bash-test.html|Bash test and comparison functions]]. Are you confused by the plethora of testing and comparison options in the Bash shell? This tip helps you demystify the various types of file, arithmetic, and string tests so you will always know when to use test, [ ], %%[[ ]]%%, %%(( ))%%, or if-then-else  constructs.
  * [[http://linuxcommand.org/index.php|LinuxCommand.org]]. Learn the Linux command line. Write shell scripts.
  * [[http://www.gentoo.org/doc/en/articles/bash-by-example-p1.xml|Bash by example]].
  * [[http://www.ss64.com/bash/|An A-Z Index of the Linux BASH command line]].
  * [[:charlas_definitivas_noviembre_2005#programacion_en_bash|Power Shell Scripting (Ponencia del Gul)]]. Ponencia a cargo de Fernando J. Pereda realizada durante el ciclo de charlas del GUL de Noviembre de 2005, incluye audio y video de la ponencia.

===== Screen =====
  * [[http://suburbia.sindominio.net/spip.php?article147|GNU Screen: Introducción y tutorial para principiantes]]. La mayoría de los sistemas Unix modernos (Linux, MacOSX y *BSD) vienen con una pequeña utilidad de consola llamada GNU-Screen. Es una herramienta poderosa en manos de guerrilleros del modo texto, una navaja suiza versátil para todo tipo de interacción persona-máquina basada en texto.
  * [[http://aperiodic.net/screen/|GNU screen]]. Página con enlaces interesantes a documentación de screen.
  * [[http://www4.informatik.uni-erlangen.de/~jnweiger/screen-faq.html|Screen FAQ]]. This is a list of frequently asked questions on screen@uni-erlangen.de. 
  * [[http://www.linuxjournal.com/article/6340|Power Sessions with Screen]]
  * [[http://www.oreillynet.com/linux/cmd/cmd.csp?path=s/screen|Alphabetical Directory of Linux Commands:screen]]
  * [[http://www.pixelbeat.org/lkdb/screen.html|linux screen keyboard shortcuts]]

===== Ssh/Scp =====
  * [[http://en.jakilinux.org/apps/ssh-tricks/|Trucos SSH]]. SSH (secure shell) es un programa que permite un acceso remoto seguro. No todo el mundo conoce el potencial de SSH, como acceso sin contraseña, ejecucion automatica de comandos en maquinas remotas, e incluso montar sistemas de ficheros a traves de SSH. En este articulo se echa un vistazo a estas caracteristicas y a muchas mas.

===== Wine =====
  * [[http://bulma.net/body.phtml?nIdNoticia=1164|Ejecuta tus programas windows en Linux con WINE]]. Wine permite ejecutar aplicaciones windows, en sistemas linux.
  * [[http://www.playonlinux.com/es/|Playonlinux]]. Playonlinux es un programa pensado originalmente para facilitar la instalación de programas de Windows en GNU/Linux mediante la capa de abstracción Wine, haciendo especial énfasis en el terreno de los juegos.
  * [[http://www.wine-doors.org|Wine-Doors]]. Wine-doors is an application designed to make installing windows software on Linux, Solaris or other Unix systems easier.
  * [[http://frankscorner.org/|Frank's Corner]]. This website contains all the information you need to get some popular Windows applications and games running on Linux using Wine.
  * [[http://www.von-thadden.de/Joachim/WineTools/|WineTools]]. WineTools is a menu driven installer for installing about 90 Windows programs under the x86 (Athlon or Intel PC) processor architecture with the Linux operating system using Wine.
