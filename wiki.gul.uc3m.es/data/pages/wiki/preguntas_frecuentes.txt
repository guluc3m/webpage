===== Preguntas frecuentes=====

Esto pretende ser una recopilación de las preguntas frecuentes que le pueden surgir a alguien que no esté familiarizado con el software libre.

  * ¿Qué es el software libre?
    El software libre es todo aquel cuya licencia permite cuatro libertades: de copia y redistribución, de estudio, de uso y de modificación. Puedes copiarlo si quieres, redistribuirlo a quien quieras por el precio que quieras, puedes estudiar su código para ver cómo funciona, puedes modificarlo a tu antojo y puedes usarlo para lo que quieras. Alguna de estas cosas te parecerá obvia, pero si lees alguna de esas licencias que todo el mundo acepta sin leer, verás que actualmente no lo son. En la página web del [[http://www.gnu.org/home.es.html|proyecto GNU]] tienes una descripción más canónica.

  * ¿el software libre es gratis?
    El software libre no tiene por qué ser gratis. En la práctica la gran mayoría de aplicaciones se distribuyen gratuitamente, pero no tiene por qué ser así. La diferencia entre comprar software privativo y software libre es que cuando compras software libre te conviertes en su poseedor, mientras que con el software privativo, generalmente, sólo tienes permiso para usarlo bajo ciertas condiciones. Cuando compras un programa de software libre puedes copiarlo, regalar las copias, vender las copias, adecuarlo a tus necesidades, basarte en ello para hacer otra cosa y todo lo que te permitan las cuatro libertades.

  * ¿Qué es Linux?
   Normalmente utilizamos Linux como abreviatura de GNU/Linux, que es un [[http://es.wikipedia.org/wiki/Sistema_operativo|sistema operativo]]. Es decir, son todos los programas que necesita un ordenador para ser útil. En realidad el [[http://www.kernel.org|proyecto Linux]] lo que trata de desarrollar es el núcleo de este sistema operativo, es decir, el programa que hace que el resto de programas puedan interactuar con los circuitos del ordenador. Puedes ver en la wikipedia la historia de [[http://es.wikipedia.org/wiki/Linux|Linux]].
 
  * Entonces ¿Qué es GNU?
  El [[http://www.gnu.org/home.es.html|proyecto GNU]] fue el primer proyecto que trató de hacer un sistema completo de [[http://www.gnu.org/philosophy/free-sw.es.html|software libre]]. Se basaron en los ampliamente utilizados sistemas [[http://es.wikipedia.org/wiki/Unix|Unix]] e hicieron un clon de ellos. Se dedicaron a rehacer las herramientas de Unix una a una hasta tener un sistema prácticamente completo. Hoy en día el proyecto sigue en desarrollo incrementándose con programas que cubran las nuevas de necesidades de los usuarios y también a falta de un núcleo. El núcleo del sistema GNU se llama [[http://es.wikipedia.org/wiki/Hurd|GNU/Hurd]] y aún está en desarrollo. Así, lo que normalmente se denomina Linux es un sistema GNU con un núcleo Linux.
 
  * ¿Qué es una distribución?
    Lo que utilizamos normalmente en el ordenador es más que un simple sistema operativo. Prácticamente cualquier programa que uses directamente, como para hablar por internet, navegar, ver vídeos, escuchar música, etc... No son parte del sistema operativo, sino que cada uno es un proyecto distinto que apenas tiene que ver con los demás. Hay grupos de gente y empresas que se dedican a coger un núcleo Linux, un sistema GNU, un montón de programas para un montón de cosas y hacer que funcionen juntos. El resultado es lo que se denomina una [[http://es.wikipedia.org/wiki/Distribuci%C3%B3n_Linux|distribución]].
